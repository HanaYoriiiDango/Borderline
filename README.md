# Borderline - Psychological Game Core

## Первый запуск

1. **Склонируйте проект**
2. **Запустите `first_run.bat`** (двойной клик)
3. **Откройте `.sln` в Visual Studio**
4. **Нажмите F5** - готово!

## Если возникли проблемы

### Русский текст не отображается:
- Убедитесь, что в Pre-build event есть: `call "$(ProjectDir)remove_bom.bat"`
- Перезапустите Visual Studio

### Bat-файл не запускается:
- Правой кнопкой → "Запуск от имени администратора"
- Или добавьте исключение в антивирус

## Команды в игре
- `help` - список команд
- `status` - состояние персонажа  
- `start` - начать диалог
- `go` - перемещение между мирами
- `exit` - завершить игру
